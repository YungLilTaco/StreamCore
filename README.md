# StreamVault (Next.js)

Modern StreamVault homepage built with Next.js (App Router), Tailwind CSS, Lucide icons, and Framer Motion.

## Getting started

1. Install **Node.js LTS** (so `npm` is available).
2. In this folder:

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Legacy static prototype (optional)

If you want to open the older static dashboard prototype, it lives in `legacy/`:

- Open `legacy/index.html` directly in your browser.
